2048

2048 is a popular single-player sliding block puzzle game where the objective is to combine tiles with the same numbers to reach the tile with the number 2048. The game is won when the player reaches the 2048 tile, but it can continue beyond that point. The game is over when the player is unable to make any more moves. You can move tiles using W, A, S and D key

I wrote this game in C language (but I am planning to rewrite it in Rust) using freeglut library.
